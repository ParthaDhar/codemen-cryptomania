    Welcome to Cryptomania Exchange Pro

    Introduction

    Cryptomania Exchange Pro is built with laravel. All the codes supported by laravel and php will also work here. It is a trading platform to build a super fast cryptocurrency trading platform for business

    Technology used

    • Php 7.2+
    • MySql 5.7+
    • Laravel 5.6+
    • Cache Driver File
    • Redis Server
    • Laravel Echo Server
    • Supervisor

    Features

    • Coinpayments Coin Api Integrated
    • BTC node Integrated
    • Exchange Scroller
    • Instant Payment notice (IPN) Ready
    • Unlimited Coin adding ability
    • Unlimited Coin Pair Creation
    • CryptoCoin Creation
    • Withdrawal and deposit histories
    • Wallet report
    • Order report
    • Buy, Sell, Stop Limit
    • Live candlestick chart
    • Moving average lines
    • Live MACD Graph
    • Exchange report
    • Google 2FA Authentication

    • Mobile Ready
    • Ultra Fast Trading Algorithm
    • Easy to go User Experience
